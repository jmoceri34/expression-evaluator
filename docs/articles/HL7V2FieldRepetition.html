Description

Field repetitions are defined in hl7 v2 specification. They exist within fields, outside of special cases such as MSH.1 and MSH.2. Field repetitions contain components.

Properties

  • Id: The position of the field repetition.
  • Delimiter: The delimiter of the field repetition.
  • Value: The value of the field repetition.
  • Components: The components on the field repetition.

Methods

  • Rebuild: Rebuilds the value of this field repetition.
  • AddComponent: Adds a component to the end of the field repetition.
  • RemoveComponent: Removes a component by Id from the field repetition.
  • InsertComponent: Inserts a component by Id into the field repetition.
  • UpdateComponent: Updates a component by Id in the field repetition.

An indexer is exposed to allow access to this field repetitions HL7V2Components.
