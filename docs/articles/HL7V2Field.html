Description

Fields are defined in hl7 v2 specification. By default they contain a single field repetition to represent the field, outside of special cases, such as the first two fields of any special segment headers (MSH.1 and MSH.2). There are CRUD operations you can perform on components and field repetitions. Field repetitions and components are both by Id, with components defaulting to the first field repetition.

Properties

  • Id: The position of the field.
  • Delimiter: The delimiter used when generating the value when calling Rebuild.
  • Value: The value of the field.
  • FieldRepetitions: The field repetitions, if any.

Methods

  • GetFieldRepetition: Gets a field repetition by Id.
  • Components: Gets this fields components by field repetition Id.
  • GetComponent: Gets this fields components by field repetition Id.
  • Rebuild: Rebuilds the Value on this field.
  • AddComponent: Adds a component to the end of this field by field repetition Id.
  • RemoveComponent: Removes a component from this field by Id and field repetition Id.
  • InsertComponent: Inserts a component into this field by Id and field repetition Id.
  • UpdateComponent: Updates a component in this field by Id and field repetition Id.
  • AddFieldRepetition: Adds a field repetition to the end of the field.
  • RemoveFieldRepetition: Removes a field repetition from the field by Id.
  • InsertFieldRepetition: Inserts a field repetition by Id into the field.
  • UpdateFieldRepetition: Updates a field repetition by Id in the field.

An indexer is exposed to access this field's HL7V2Components by their Ids and repetition (default is 1).
