Description

Components are defined in hl7v2 specification. They can contain sub components and exist within field repetitions.

Properties

  • Id: The position of the component.
  • Delimiter: The delimiter of the component.
  • Value: The value of the component.
  • SubComponents: The sub components of the component.

Methods

  • GetSubComponent: Gets a sub component by Id.
  • Rebuild: Rebuilds this components Value.
  • AddSubComponent: Adds a sub component to the end of the component.
  • RemoveSubComponent: Removes a sub component by Id.
  • InsertSubComponent: Inserts a sub component by Id.
  • UpdateSubComponent: Updates a sub component by Id.

An indexer is exposed to allow access to this components HL7V2SubComponents.
